## 📖 What is SijanList?

**SijanList** is a free, lightweight Chrome extension that replaces your New Tab with a beautiful and smart bookmark dashboard. Instead of a blank tab, every new tab shows your bookmarks — organized into folders, boards, and pages — so you can access everything you need in one click.

> **Live since 26 March, 2026 · Built by [Sijan](https://sijan.pro.bd/)**

---

## ✨ Features

| Feature | Description |
|---|---|
| 📄 **Tab Pages** | Create multiple pages — Home, Work, Personal — and switch instantly |
| 📋 **Board System** | Organize bookmarks into custom boards and categories |
| 🔖 **Bookmark Manager** | Add, edit, delete, and drag-to-reorder bookmarks |
| 🗑️ **Trash & Restore** | Accidentally deleted? Restore with one click |
| ⚡ **Fast & Lightweight** | Pure HTML, CSS, Vanilla JS — loads in milliseconds |
| 🔒 **Zero Tracking** | No analytics, no telemetry, no data collection — ever |
| 🎨 **Fully Customizable** | Custom backgrounds, colors, icons, and layouts |
| 📡 **Works Offline** | Everything runs locally — no internet needed |
| 🌐 **Multi-Browser** | Works on Chrome, Edge, Brave, and Opera |

## 🚀 Installation

### Method 1 — Direct Download (Recommended)

1. Go to **[sijanlist.pro.bd](https://sijanlist.pro.bd/)** and click **Download Free**
2. Extract the downloaded ZIP file
3. Open Chrome → go to `chrome://extensions`
4. Enable **Developer Mode** (top right toggle)
5. Click **"Load unpacked"** → select the extracted folder
6. Open a new tab — enjoy your smart dashboard! 🎉

### Method 2 — Clone this repo

```bash
git clone https://github.com/seizmann/sijanlist.git
```

Then follow steps 3–6 above.

## 🛠️ Tech Stack

- **Frontend** — HTML5, CSS3, Vanilla JavaScript
- **Storage** — Chrome Extension Storage API (`chrome.storage.local`)
- **Backend** — Supabase (download stats tracking)
- **Hosting** — [sijanlist.pro.bd](https://sijanlist.pro.bd/)
- **No frameworks** — intentionally lightweight
